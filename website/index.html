Cross-model consensus

Each suggestion is annotated with the share of models that agreed and the cluster of overlapping spans. False positives drop sharply when a fix has 11/13 backing it.

Bring your own keys

API keys live on your device. The local server uses them transiently for each request and never stores them. Run it on your laptop or self-host the Docker image.

Featherless community key

Tip the project to unlock a shared Featherless.ai key — hundreds of open-weight models behind one endpoint. No subscription needed.

Code-aware corpus

Every accepted correction is appended to a JSONL corpus with detected programming language. Fine-tune your own internal model from it on Hugging Face.

Works everywhere you write

Browsers via the extension. Word, Pages, Outlook, Mail, and any AX-compliant app via the Mac app. Press ⌥⌘G and Qrammerly takes over.

Tone, goals, readability

Beyond grammar: tone detection, audience- and formality-aware proofreading, Flesch readability, vocabulary diversity, reading time.
