# Qrammarly — Microsoft Edge

Edge is Chromium-based and consumes the same MV3 manifest as Chrome.

## Install (developer)

1. Open `edge://extensions`.
2. Toggle on **Developer mode**.
3. Click **Load unpacked** and select this folder.

## Publishing

Submit to the Microsoft Edge Add-ons store via
[Partner Center](https://partner.microsoft.com/dashboard/microsoftedge/).
